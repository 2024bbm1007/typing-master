# How to Use Keyboard Shortcuts While Coding

Keyboard shortcuts speed up coding and reduce mouse fatigue. This post shows the most useful shortcuts for common editors.

- **VS Code**: `Ctrl+Shift+P` (Command Palette), `Ctrl+P` (Quick Open), `Ctrl+Space` (IntelliSense).
- **IntelliJ**: `Ctrl+Shift+A` (Find Action), `Ctrl+N` (Navigate to Class).
- **General**: `Ctrl+Z` (Undo), `Ctrl+Y` (Redo), `Ctrl+F` (Find).
